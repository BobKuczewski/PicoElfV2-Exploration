# PicoElfV2-Exploration
Software and Results from exploring a Pico Elf Version 2

## Introduction
This repository will contain a mixture of software and results from exploring a Pico Elf Version 2 (PE2). This will include understanding how operating systems are installed, the format of the file system, how to write programs for the PE2.

Beware that because this project is an exploration, some of the information posted here may be incorrect.
